This is a 7day prototype for For the next seven days, create one project per day. Each project must be conceptualized, produced and documented in a single day.


Day 1 Tuesday

Platform Testing
Test if unity is a good platform to work on and make game within a feasible time constraint. I set out to learn Unity basics and made a small platform testing. It is game which has a ball that collects pickups and winning objective is to collect them all.


Day 2 Wednesday

Theme and Mapping a Game
I thought about the game narrative and theme. I spent my time imagining walking through differengt scenarios and decided on an island theme for the game designed like an escape room. I drew sketches for maps and scenes inside the game.
The game will start off with the character flying into the middle of the terrain. The winning objective will be finding the oars to a boat to escape.

Day 3 Thursday

Terrain Mapping on Unity
From the sketches I made a basic terrain in Unity.
REFLECTION: I felt like I create a terrain which is very big and instead of looking at the big picture of things I should focus on smaller aspects.

Day 4 Friday

Game Flow
Escape Room situated on an island having 4 different levels- one at beach, one at a treehouse, one by the water stream and one at a temple. I made a game flow having selected different components for the game.
REFLECTION: I thought about different ways to use these components that can be found in the terrain.

Day 5 Saturday

Analyse Games and their flow, study puzzles in escape rooms.
I thought about what puzzles are in different games and what they can teach when you solve each puzzle. I looked at various sources of puzzle designing and construction, and made notes.

Day 6 Sunday

Day 7 Monday

Check Color Blind filters and make color palette
One of my game traits focuses on color blind modes in technology so I studied color blind modes on a game instance and made a color palette that would work in the different modes.
